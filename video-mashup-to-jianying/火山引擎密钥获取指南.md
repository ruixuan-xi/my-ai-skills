# 火山引擎三个密钥获取指南

本文档详细说明如何获取以下三个火山引擎参数：
- **ArkAPIKey**：方舟大模型 API Key（用于文生图 / 视觉理解 / LLM 对话）
- **TTSAppID**：语音合成（TTS）应用 ID
- **TTSAccessToken**：语音合成（TTS）访问令牌

> 一句话概括：**ArkAPIKey 在"火山方舟"控制台申领；TTSAppID / TTSAccessToken 在"语音技术"控制台的应用管理里领取**。两者属于不同产品线，入口不一样，别搞混。

---

## 一、获取 ArkAPIKey（火山方舟 API Key）

> 火山引擎已预置推理接入点，直接申领 API Key 即可调用，**无需自行创建 Endpoint**。

### 步骤

| 步骤 | 操作 |
|:---:|:---|
| 1 | 控制台首页搜索 **"火山方舟"**，或直接访问 <https://console.volcengine.com/ark> |
| 2 | 左侧菜单 → **API Key 管理** |
| 3 | 点击 **"创建 API Key"** 按钮 |
| 4 | 填写名称（如 `volcengine-api`），提交 |
| 5 | 复制生成的 Key（形如 `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`）—— **这就是 ArkAPIKey** |

### 直达链接

- API Key 管理：<https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey>

---

## 二、获取 TTSAppID 和 TTSAccessToken（语音合成）

⚠️ **注意**：TTS 的密钥**不在**火山方舟里，而在"**语音技术**"这个独立产品中。

### 步骤

| 步骤 | 操作 |
|:---:|:---|
| 1 | 控制台首页搜索 **"语音技术"**，或访问 <https://console.volcengine.com/speech/app> |
| 2 | 左侧菜单 → **应用管理** |
| 3 | 点击 **"创建应用"** |
| 4 | 填写应用名（如 `volcengine-tts`），**勾选功能**：`语音合成`、`大模型语音合成` 都勾上 |
| 5 | 提交后，打开链接：<https://console.volcengine.com/speech/service/10047?AppID=8113635306>（把末尾的 `AppID` 换成你自己应用的 ID），页面上即可看到 **APP ID** 和 **Access Token** |

### 两个值的位置

在应用详情页你会看到：

```
APP ID       : 1234567890         ← 这个填入 TTSAppID
Access Token : xxxxxxxxxxxxxxxxx  ← 这个填入 TTSAccessToken
```

### 直达链接

- 应用管理：<https://console.volcengine.com/speech/app>
- 大模型语音合成服务详情（含 AppID / Token 展示）：<https://console.volcengine.com/speech/service/10047?AppID=8113635306>
- 计费说明：<https://www.volcengine.com/docs/6561/1257544>

## 三、Python SDK 安装提示

调用火山引擎 Python 接口时，如果报：

```
No module named 'volcenginesdkarkruntime'
```

根据火山引擎官方文档和社区解决方案，需要安装的包是：

```
pip install volcengine-python-sdk[ark]
```
